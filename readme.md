
## ScreenDraw 1.0 For Windows

Simple screen draw application. Draw directly on screen and make a screen shoot to save results


### Features 
* Draw on screen
* Twelve colors
* Save to desktop

### Sources:
This application was builded with Visual Studio 2015, C# and WPF
